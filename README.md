# Projet Stock Tracker

Ce projet est une application web qui permet de suivre les cours des actions. Il est composé d'un backend en Python et d'un frontend en React.

## Backend

Le backend est construit avec FastAPI et fournit une API pour récupérer les données boursières.

Pour plus d'informations, consultez le `README.md` dans le dossier `backend`.

## Frontend

Le frontend est une application React qui consomme l'API du backend pour afficher les données boursières sous forme de graphiques.

Pour plus d'informations, consultez le `README.md` dans le dossier `frontend`.

## Démarrage

Pour lancer le projet, vous devez démarrer le backend et le frontend séparément.

### Backend

```bash
cd backend
# ... instructions du README du backend ...
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

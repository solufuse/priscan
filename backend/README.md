# Backend

Ce dossier contient le code du backend de l'application.

## Framework

Le backend est construit avec [FastAPI](https://fastapi.tiangolo.com/), un framework web moderne et performant pour Python.

## Démarrage

Pour lancer le serveur de développement, exécutez les commandes suivantes :

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

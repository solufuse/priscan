# Frontend

Ce dossier contient le code du frontend de l'application, construit avec Vite, React et TypeScript.

## Bibliothèques et Outils

- **Vite**: Pour un développement rapide et un build optimisé.
- **React**: Pour la construction de l'interface utilisateur.
- **TypeScript**: Pour un code plus sûr et plus maintenable.
- **Tailwind CSS**: Pour le style de l'application.
- **Shadcn/ui**: Pour une collection de composants d'interface utilisateur réutilisables et accessibles.
- **Lightweight Charts**: Pour l'affichage des graphiques boursiers.

## Démarrage

Pour lancer le serveur de développement, exécutez les commandes suivantes :

```bash
cd frontend
npm install
npm run dev
```
